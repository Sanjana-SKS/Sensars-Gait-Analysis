# React + TypeScript + Vite

### Setup Instructions for Team Members
1️⃣ Clone the Repository
git clone https://github.com/Sanjana-SKS/Sensars-Gait-Analysis.git
cd Sensars-Gait-Analysis

2️⃣ Install Dependencies
npm install

3️⃣ Set Up Environment Variables
The Firebase environment variables are required to run the project. The .env file has already been created and shared separately. Ensure you have it in the root directory.

4️⃣ Run the Development Server
npm run dev
Your app should now be live at: http://localhost:5173/
